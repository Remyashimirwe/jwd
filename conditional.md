

## 1 Basic `if`

Write JavaScript code that checks if a number **age** is greater than or equal to **18**.
If it is, print:

```
"You can vote"
```

---

## 2`if...else`

Write code that checks a variable **temperature**.

* If temperature is greater than **30**, print `"It is hot"`
* Otherwise print `"Weather is normal"`

---

## 3️ `if...else if...else`

Write JavaScript code that checks a variable **marks** and prints:

* `"Distinction"` if marks ≥ 80
* `"Pass"` if marks ≥ 50
* `"Fail"` if marks < 50

---

## 4️ Even or Odd

Write JavaScript code that checks if a number **num** is even or odd using **if...else**.

Example output:

```
"Even number"
```

or

```
"Odd number"
```

---

## 5️ Positive, Negative, or Zero

Write code that checks a number **n** and prints:

* `"Positive"` if n > 0
* `"Negative"` if n < 0
* `"Zero"` if n == 0

# Nested `if` Questions

## 6️ Login Check

Write code that checks:

If **username == "admin"**

Then check if **password == "1234"**

* If correct → print `"Login successful"`
* Otherwise → `"Wrong password"`

If username is wrong → `"User not found"`

---

## 7️ Student Eligibility

Write a **nested if** program.

If **age ≥ 18**

Then check if **marks ≥ 50**

* If true → `"Eligible for admission"`
* Otherwise → `"Marks too low"`

Else → `"Too young"`

---
